Pastebin Buddy
 
←back to portfolio

This was my very first major project in the Python 3 programming language (so please try and excuse the bad programming). It's purpose is simple: to interact with the Pastebin API to allow for scraping and posting (both anonymously or with an account).

It has seen practical use. For example, at one point I was doing live scraping of statistics of users on Instagram and analyzing the data to determine whether or not their follower gains were legitimate. Whilst all the work could be done on one computer, I decided that being able to see the numbers on my phone anytime, anywhere, would be a both fun to implement and an entertaining thing to have. This is where I used Pastebin Buddy to upload the live scraped data to the Pastebin account, and was able to check the information using the Pastebin app.

Despite its simplicity, this project has been both educating and fun to do. The final product is also quite useful.
